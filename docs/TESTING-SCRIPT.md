# Pathoro v0.1 — Testing Script

**Production URL:** https://pathoro-mvp.vercel.app

This is a lightweight script for running informal user tests on the v0.1 live MVP. Sit with the tester, share the link, and let them click through without guiding them — take notes on where they hesitate or ask questions.

## Test flow

Ask the tester to go through the following steps out loud (think-aloud style):

1. Land on the homepage
2. Understand the orientation
3. Continue to route planning
4. Understand the five routes
5. Click "Take this step"
6. Understand the opportunity detail
7. Go back to opportunities

## Main question

Ask this before any follow-ups, right after they've clicked through:

> "What do you think this product helps someone do?"

## Follow-up questions

- "Where did you feel most clear?"
- "Where did you feel confused?"
- "Did the opportunity feel real/reachable?"
- "Did the route idea make sense?"
- "What would you expect to happen next?"
- "Would this be more useful with real local opportunities?"
- "Would you want to post an opportunity here?"
